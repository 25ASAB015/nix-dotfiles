## MCP Servers (Model Context Protocol)

Los **MCP Servers** extienden las capacidades de OpenCode proporcionando contexto adicional y herramientas especializadas.

### MCP Servers Configurados

1. **gh_grep**: B칰squeda en GitHub
   - Tipo: `remote`
   - URL: `https://mcp.grep.app/`
   - Timeout: 10000ms

2. **deepwiki**: B칰squeda en DeepWiki
   - Tipo: `remote`
   - URL: `https://mcp.deepwiki.com/mcp`
   - Timeout: 10000ms

3. **context7**: Contexto adicional
   - Tipo: `remote`
   - URL: `https://mcp.context7.com/mcp`
   - Timeout: 10000ms

4. **context-manager**: Gestor de contexto local
   - Tipo: `local`
   - Comando: `["bunx" "mcp-context-manager"]`
   - Requiere: `bunx` (Bun package runner)

### Agregar un Nuevo MCP Server

En `hydenix-config.nix`:

```nix
modules.terminal.software.opencode = {
  mcp = {
    mi-mcp = {
      type = "remote";  # o "local"
      url = "https://mcp.example.com/";
      enabled = true;
      timeout = 10000;
    };
  };
};
```

Para servidores locales:

```nix
mi-mcp-local = {
  type = "local";
  command = ["comando" "argumentos"];
  enabled = true;
};
```

**Nota**: Los servidores locales usan `command` en lugar de `url`, y no requieren `timeout`.
