# Making your own theme

Si quieres crear tu propio tema, el flujo recomendado es basarte en los temas
de la galeria oficial y seguir la guia de HyDE para theming. [HyDE Project - hyde-gallery](https://github.com/HyDE-Project/hyde-gallery.git)

Puntos clave:

- Usa un tema existente como base.
- Mantiene consistencia entre wallpaper, Waybar, launcher y lockscreen.
- Comparte tu tema via `hydectl theme import --url ...` o contribuyendo a la galeria.
