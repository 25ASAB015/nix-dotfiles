# Monitors

Esta página detalla la configuración de monitores en el sistema.

## Configuración Física Actual

El sistema está configurado con dos monitores principales:

1.  **Sony TV (HDMI-A-1)**:
    *   **Posición**: Izquierda (`0x0`).
    *   **Resolución**: 1920x1080 @ 60Hz.
    *   **Uso**: Monitor auxiliar.

2.  **Dell P2417H (DP-1)**:
    *   **Posición**: Derecha (`1920x0`).
    *   **Resolución**: 1920x1080 @ 60Hz.
    *   **Uso**: Monitor principal.

## Gestión de Configuración

La configuración de los monitores se gestiona a través del archivo:
`~/Dotfiles/resources/config/hypr/monitors.conf`

Este archivo es enlazado simbólicamente a `~/.config/hypr/monitors.conf` mediante Home Manager.

### Ejemplos de Configuración

Para cambiar la posición de un monitor, edita el archivo `monitors.conf` y ajusta las coordenadas:

```bash
# Formato: monitor=ID,resolución@tasa,posición,escala
monitor=HDMI-A-1,1920x1080@60,0x0,1
monitor=DP-1,1920x1080@60,1920x0,1
```
