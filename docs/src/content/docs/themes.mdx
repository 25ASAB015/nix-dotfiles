# Themes

Esta seccion documenta el menu de HyDE en Waybar y como cambiar temas,
wallpapers y layouts de Waybar, siguiendo el estilo de la guia de Omarchy.
La idea es similar: un tema aplica un estilo coherente al entorno completo,
incluyendo aplicaciones y componentes del escritorio. [Omarchy Themes](https://learn.omacom.io/2/the-omarchy-manual/52/themes)

## Menu de HyDE en Waybar

El menu de HyDE aparece en Waybar (icono del menu). Estas son las opciones
principales y los atajos disponibles en tus dotfiles:

| Opcion | Que cambia | Atajos |
| --- | --- | --- |
| Animations | Animaciones de Hyprland | Sin atajo definido |
| Wallpaper | Seleccion y rotacion de fondos | `Super + W`, `Super + .`, `Super + ,` |
| Theme | Selector de temas | `Super + T` |
| Waybar | Layouts y visibilidad de la barra | `Super + Ctrl + .`, `Super + Ctrl + ,`, `Alt derecho + Ctrl derecho` |
| Workflows | Perfiles o flujos del entorno | Sin atajo definido |
| Lockscreen | Configuracion del bloqueo | `Super + X` |
| Shaders | Filtros/efectos visuales | Sin atajo definido |
| Keybinds | Ayuda de keybindings | `Super + /` |
| About | Informacion de HyDE | Sin atajo definido |

## hydectl (general)

`hydectl` es la CLI oficial de HyDE. Comandos disponibles hoy:

| Comando | Que hace |
| --- | --- |
| `hydectl completion` | Autocompletado para tu shell |
| `hydectl config` | Editor interactivo de configuracion |
| `hydectl dispatch` | Ejecuta comandos de plugins |
| `hydectl reload` | Recarga configuracion de HyDE |
| `hydectl select` | Selector de items (menus) |
| `hydectl tabs` | Agrupar/desagrupar ventanas del workspace |
| `hydectl theme` | Gestion de temas |
| `hydectl wallpaper` | Gestion de wallpapers |
| `hydectl zoom` | Zoom in/out en Hyprland |
| `hydectl version` | Version de hydectl |

## Cambiar tema

- Desde el menu: `Theme`.
- Atajo directo: `Super + T`.

Los temas en HyDE/Hydenix buscan mantener una estetica coherente entre
escritorio, terminal, Neovim, notificaciones, Waybar, launcher y lockscreen,
de forma parecida a lo descrito en Omarchy. [Omarchy Themes](https://learn.omacom.io/2/the-omarchy-manual/52/themes)

## hydectl (themes)

Comandos actuales para temas:

| Comando | Que hace |
| --- | --- |
| `hydectl theme select` | Selector de temas |
| `hydectl theme next` | Siguiente tema |
| `hydectl theme prev` | Tema anterior |
| `hydectl theme set <nombre>` | Aplicar tema por nombre |
| `hydectl theme import` | Importar temas adicionales |

Para instalar temas extra, revisa `Extra themes`.

## Wallpapers del tema

Si el tema incluye multiples fondos, puedes rotarlos con:

- `Super + .` siguiente wallpaper global
- `Super + ,` wallpaper global anterior
