# Hotkeys

Los atajos se generan desde los dotfiles en `~/Dotfiles`. Esta pagina centraliza
los keybindings definidos en el sistema (Hyprland), terminal y Neovim.

## Convenciones

- `Super` = tecla Windows/Meta.
- `$mainMod` = `Super`.
- `$TERMINAL`, `$EDITOR`, `$EXPLORER`, `$BROWSER` se toman de tu config de Hypr.
