# Launchers

Corresponde a la sección `Launcher` en `keybinds.conf`.

## Apps (Vim mnemonic style)

| Atajo | Función |
| :--- | :--- |
| `Super + Enter` | Terminal emulator |
| `Super + Alt + Enter` | Floating terminal (75%) |
| `Super + E` | File explorer |
| `Super + t` | Text editor |
| `Super + B` | Web browser |
| `Super + N` | Neovide |
| `Super + Y` | Yazi |
| `Super + M` | Spotify |
| `Ctrl + Shift + Escape` | System monitor |

## Rofi Menus (Vim leader style)

| Atajo | Función |
| :--- | :--- |
| `Super + Space` | Application finder |
| `Super + Tab` | Window switcher |
| `Super + Shift + E` | File finder |
| `Alt + F1` | Keybindings hint |
| `Super + ;` | Emoji picker |
| `Super + '` | Glyph picker |
| `Super + P` | Clipboard history |
| `Super + Shift + P` | Clipboard manager |
