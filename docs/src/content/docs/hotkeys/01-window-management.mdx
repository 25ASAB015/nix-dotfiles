# Window Management

Corresponde a la sección `Window Management` en `hypr/keybinds.conf`.

## Basic Controls

| Atajo | Función |
| :--- | :--- |
| `Super + Q` | Cerrar ventana (no mata Steam) |
| `Alt + F4` | Cerrar ventana (no mata Steam) |
| `Super + Shift + Space` | Alternar ventana flotante |
| `Super + G` | Alternar grupo |
| `Super + F` | Pantalla completa (estilo Vim) |
| `Super + Ctrl + F` | Pantalla completa en modo tiled |
| `Super + Alt + F` | Full width |
| `Super + Shift + Alt + F11` | Forzar pantalla completa |
| `Alt + F11` | Full width |
| `Super + X` | Bloquear pantalla |
| `Super + Alt + P` | Fijar ventana (pin) |

| `Ctrl + Alt + Delete` | Menú de logout |

## Group Navigation

| Atajo | Función |
| :--- | :--- |
| `Super + [` | Grupo anterior |
| `Super + ]` | Grupo siguiente |
| `Super + Alt + Tab` | Siguiente ventana en grupo |
| `Super + Alt + Shift + Tab` | Ventana anterior en grupo |

## Focus Navigation

| Atajo | Función |
| :--- | :--- |
| `Super + H` | Foco a la izquierda |
| `Super + L` | Foco a la derecha |
| `Super + K` | Foco arriba |
| `Super + J` | Foco abajo |
| `Super + Left` | Foco a la izquierda (Arrow) |
| `Super + Right` | Foco a la derecha (Arrow) |
| `Super + Up` | Foco arriba (Arrow) |
| `Super + Down` | Foco abajo (Arrow) |
| `Super + '` | Foco a la ventana anterior |

## Window Resize (Vim Style)

| Atajo | Función |
| :--- | :--- |
| `Super + Ctrl + L` | Aumentar ancho |
| `Super + Ctrl + H` | Disminuir ancho |
| `Super + Ctrl + K` | Aumentar alto |
| `Super + Ctrl + J` | Disminuir alto |
| `Super + Ctrl + +` | Aumentar tamaño de ventana |
| `Super + Ctrl + -` | Disminuir tamaño de ventana |
| `Super + Ctrl + Left` | Disminuir ancho (Ctrl+Arrow) |
| `Super + Ctrl + Right` | Aumentar ancho (Ctrl+Arrow) |
| `Super + Ctrl + Up` | Aumentar alto (Ctrl+Arrow) |
| `Super + Ctrl + Down` | Disminuir alto (Ctrl+Arrow) |

## Window Movement (Vim Style)

| Atajo | Función |
| :--- | :--- |
| `Super + Shift + H` | Mover a la izquierda |
| `Super + Shift + L` | Mover a la derecha |
| `Super + Shift + K` | Mover arriba |
| `Super + Shift + J` | Mover abajo |

## Swap Window

| Atajo | Función |
| :--- | :--- |
| `Super + Shift + Left` | Intercambiar con ventana izquierda |
| `Super + Shift + Right` | Intercambiar con ventana derecha |
| `Super + Shift + Up` | Intercambiar con ventana arriba |
| `Super + Shift + Down` | Intercambiar con ventana abajo |

## Move & Resize with Mouse

| Atajo | Función |
| :--- | :--- |
| `Super + Mouse 1` | Mover ventana |
| `Super + Mouse 2` | Redimensionar ventana |
| `Super + Z` | Mover ventana (teclado + mouse sim) |
| `Super + Shift + Z` | Redimensionar ventana (teclado + mouse sim) |

## Splits

| Atajo | Función |
| :--- | :--- |
| `Super + S` | Alternar split (estilo vim :sp/:vsp) |
