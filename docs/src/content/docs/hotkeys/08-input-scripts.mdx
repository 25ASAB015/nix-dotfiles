# Input & Scripts

Corresponde a las secciones finales de `hypr/keybinds.conf`: `FCITX`, `Custom Scripts` y `Screen Recording`.

## Input Method (FCITX)

| Atajo | Función |
| :--- | :--- |
| `Control + Space` | Toggle input method |
| `Super + Shift + I` | FCITX config tool |

## Custom Scripts

| Atajo | Función |
| :--- | :--- |
| `Super + Ctrl + S` | Script launcher |
| `Super + D` | Open or focus dictionary |

## Screen Recording

| Atajo | Función |
| :--- | :--- |
| `Super + Shift + R` | Toggle region recording (MP4) |
| `Super + Shift + G` | Toggle region recording (GIF) |
