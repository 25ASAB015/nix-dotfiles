# Hotkeys

Esta sección documenta los atajos de teclado configurados en Hydenix (Hyprland).

Los atajos se generan desde los dotfiles en `~/Dotfiles`.

## Convenciones

- `Super` = tecla Windows/Meta.
- `$mainMod` = `Super`.
- `$TERMINAL`, `$EDITOR`, `$EXPLORER`, `$BROWSER` se toman de tu config de Hypr.

## Contenido

Elige una categoría en la barra lateral para ver los atajos correspondientes.
