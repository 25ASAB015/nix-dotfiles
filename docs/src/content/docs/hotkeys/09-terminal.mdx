# Terminal Ecosystem

Esta sección documenta atajos que son específicos de las aplicaciones de terminal y no están controlados por Hyprland (`keybinds.conf`).

## Ghostty

| Atajo | Función |
| :--- | :--- |
| `Ctrl + Shift + I` | Toggle inspector |
| `Ctrl + Shift + P` | Command palette |
| `Ctrl + Shift + C` | Copiar |
| `Ctrl + Shift + V` | Pegar |
| `Ctrl + Shift + +/-/0` | Control de tamaño de fuente |
| `Ctrl + Shift + R` | Recargar configuración |
| `Alt + V/S` | Splits (Derecha/Abajo) |
| `Alt + H/J/K/L` | Navegación entre splits |
| `Alt + N` | Nueva pestaña |

## Fish Shell

| Atajo | Función |
| :--- | :--- |
| `Ctrl + Backspace` | Borrar palabra atrás |
| `Ctrl + Z` | Undo |
| `Ctrl + B/E` | Inicio/Fin de línea |
| `Flecha Arriba/Abajo` | Búsqueda en historial (prefix search) |
| `Ctrl + X` | Atajo de `fifc` |
| `Ctrl + S` | Secuencia de `sudope` |

## Atuin

| Atajo | Función |
| :--- | :--- |
| `Ctrl + R` | Abrir historial (Atuin) |
| `Ctrl + N` | Atajos nativos de Atuin UI |
