# Wallpaper & Theming

Corresponde a la sección `Theming and Wallpaper` en `keybinds.conf`.

Todos los atajos visuales están unificados bajo el modificador `Super + Alt`.

## Selectors (Menús)

| Atajo | Función |
| :--- | :--- |
| `Super + Alt + W` | **W**allpaper Select (Rofi) |
| `Super + Alt + T` | **T**heme Select (Rofi) |
| `Super + Alt + B` | Waybar **B**ar Styles (Layout) |
| `Super + Alt + R` | Wallbash **R**ice Mode |

## Navigation (Cycle)

| Atajo | Función |
| :--- | :--- |
| `Super + Alt + ]` | Siguiente Wallpaper |
| `Super + Alt + [` | Anterior Wallpaper |

## Toggles & Options

| Atajo | Función |
| :--- | :--- |
| `Super + Alt + O` | Toggle **O**pacity (On/Off) |
| `Super + Alt + G` | **G**ame Mode (Desactiva animaciones/efectos) |
