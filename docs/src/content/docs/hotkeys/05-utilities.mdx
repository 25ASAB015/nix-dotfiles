# Utilities

Corresponde a la sección `Utilities` en `hypr/keybinds.conf`.

## General Utilities

| Atajo | Función |
| :--- | :--- |
| `Super + Backslash` | Toggle keyboard layout |

| `Super + Alt + G` | Open game launcher |
| `Alt_R + Control_R` | Toggle Waybar and reload config |
| `Control + Escape` | Reload Hyprland |

## Screen Capture

| Atajo | Función |
| :--- | :--- |
| `Alt + Print` | Color picker |
| `Super + Shift + S` | Snip screen |
| `Super + Shift + W` | Window screenshot (edit) |
| `Super + Ctrl + Shift + S` | Freeze and snip screen |
| `Shift + Print` | Snip screen to clipboard |
| `Super + Print` | Print all monitors |
| `Print` | Print monitor |
