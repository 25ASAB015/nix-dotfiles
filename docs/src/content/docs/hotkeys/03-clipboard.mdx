# Clipboard

Corresponde a la sección `Clipboard universal` en `keybinds.conf`.

## Universal Shortcuts

Estos atajos utilizan `sendshortcut` para enviar las combinaciones de teclas tradicionales (Ctrl+Insert, etc.) a aplicaciones que no soportan los atajos estándar o para unificar el comportamiento.

| Atajo | Función | Descripción |
| :--- | :--- | :--- |
| `Super + C` | Copiar | Envía `CTRL + Insert` |
| `Super + V` | Pegar | Envía `SHIFT + Insert` |
| `Super + X` | Cortar | Envía `CTRL + X` |

> *Nota: Para el historial de portapapeles, ver la sección de Rofi en Launchers (`Super + P`).*
